Everything in the studio

Every project, grouped by what it is — each an instance of the same idea: an AI model is a powerful but unreliable proposer, wrapped in a deterministic scaffold that decides what's allowed to be true.

The model proposes; the scaffold disposes. Knowledge, story, video, music, code, games, play — the model fills it in, deterministic gates decide what survives.
