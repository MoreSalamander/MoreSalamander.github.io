The shape

Free agents at the edges, one incorruptible gate at the center.

Scouts and verifiers operate in an unconstrained proposal zone — they can be wrong, they can speculate, they can argue. Nothing they produce is trusted. The scaffold zone is exactly one gate, and it fails closed: known-scam domains hard-reject anywhere, unresolvable evidence is excluded rather than trusted, and only the gate's own verdict is ever persisted as truth.

Constitution

Hard invariants, every phase.

Never holds keys

Never signs or executes a transaction. Information system only.

Deterministic trust boundary

Nothing reaches the user except through the fail-closed gate.

Provenance, always

Every trusted claim carries a source URL, timestamp, and confirmation count.

Intelligence, not advice

"Verified, costs X, expires Y" — never "put your money here."

The org

Fifteen agents, four layers, one debate.
LayerDoes
ScoutsSix beats — airdrops, quests, testnets, NFTs, governance, staking — file candidate opportunities from curated live sources
VerificationSource Verification + Scam Detection agents gather evidence; the gate alone consumes it and rules
IntelligenceRanking Agent scores within a deterministic rubric; Pattern Detection mines outcome history
StrategyProfile-aware Daily Mission Agent builds tonight's plan against your real time and money budget

Before anything reaches the mission, the debate contests it — Advocate, Skeptic, and Strategist file stances (endorse / caution / oppose). Stances never change the verdict; only new evidence from the Skeptic can trigger the gate to re-rule.
